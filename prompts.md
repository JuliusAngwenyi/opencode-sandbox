# Prompts Used

## CLI Calculator
> "code a minimal cli calculator in python"

Generated `calc.py` - a minimal command-line calculator using Python's `eval()` with basic input validation.

## Snake Game
> "Please code a Snake Game in Python using pygame."

Generated `snake.py` - a classic Snake game using pygame with arrow key controls and score tracking.

## Sudoku Browser Game
> "Code a soduku browser game. The game should have a button for generating a soduku and a dropdown for selecting the difficulty level. Then the soduku shall be displayed, the user shall be able to play it. With all the rules that are usual. Make sure it works completely and is well-designed. Do not use external packages for the soduku logic. Implement the algorithms yourself for all necessary parts. Use the following tech stack: Python Backend, HTML JS CSS Frontend"

Generated `sudoku/` - a full-stack Sudoku browser game with Python/Flask backend and HTML/CSS/JS frontend. Includes custom Sudoku generator, solver, and validator with Easy/Medium/Hard difficulty levels.
