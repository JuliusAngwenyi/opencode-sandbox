# OpenCode Sandbox

A collection of small Python projects including a CLI calculator and a Snake game.

## Projects

### CLI Calculator (`calc.py`)
A minimal command-line calculator that evaluates arithmetic expressions.

```bash
python3 calc.py 2 + 3
python3 calc.py "(2 + 3) * 4"
```

### Snake Game (`snake.py`)
A classic Snake game built with pygame.

**Controls:** Arrow keys to move, R to restart after game over.

```bash
pip install pygame
python3 snake.py
```

## Getting Started

### Prerequisites
- Python 3.6+
- pip (for Snake game dependencies)

### Installation
1. Clone the repository:
```bash
git clone https://github.com/JuliusAngwenyi/opencode-sandbox.git
cd opencode-sandbox
```

2. Install dependencies (for Snake game):
```bash
pip install pygame
```

### Running
**Calculator:**
```bash
python3 calc.py <expression>
```

**Snake Game:**
```bash
python3 snake.py
```
